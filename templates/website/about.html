The main objective of this project was first and foremost to practice web development.
Development Stack:
 Database: MySQL 8.0
 Backend: Django/Python
 Frontend: Bootstrap and JQuery
 Deployment: Google App Engine (GAE)
 VCS: GitHub (More @ LINK TO GITHUB PROJECT)

Secondly, to create a database of professional artists in general, aiming at Machine Learning/Pattern Recognition to relate similar images, using algorithms such as TF-IDF, KNN and Dot Product. The items are registered using an underlying system. Unfortunately the database does not have enough examples to implement a CNN yet, although once it reaches an considerable size it will be done and all the data will be posted on Kaggle.

The database was designed by myself. An image of the system model can be found at LINK_TO_ERR_IMAGE. (You can find a little bit strange, but i did not use the Django User Model for the login.)

This project is currently being reworked, with a different stack of technologies, such as Docker, DjangoREST.




If you would like to publish your work here. You can use this contact form to send a link to where the files are stored at. The more the detail about the art the better, such as materials used, in what/who it was inspired, the title and description for instance.
Remember that it can be anything, such as sculptures, tatoos, drawings, paintings etc.
